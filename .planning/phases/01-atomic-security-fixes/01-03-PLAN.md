**BUG-02: Fix buildSignedTransaction**

7. The function signature already accepts `publicKey: Uint8Array` (line 323). Fix the body to include publicKey bytes between key type and signature:
```typescript
function buildSignedTransaction(
  transaction: Uint8Array,
  signature: Uint8Array,
  publicKey: Uint8Array
): Uint8Array {
  const parts: Uint8Array[] = [];

  // Transaction bytes
  parts.push(transaction);

  // Signature enum (ED25519 = 0)
  parts.push(new Uint8Array([0]));           // keyType: 1 byte
  parts.push(new Uint8Array(publicKey));     // publicKey: 32 bytes
  parts.push(new Uint8Array(signature));     // signature data: 64 bytes

  return concatArrays(parts);
}
```

The current code (lines 330-332) is missing the `publicKey` push. Add it between the key type byte and the signature bytes.
